# STlinkV3-Adapter
Custom developed board that allow you to use the new STLINK-V3Mini.<br>
This adapter board was designed to retain the STDC14 connector from stlinkv3-mini.

**SWD connector**
|Pin Number|Description|
|----------|-----------|
|       1  | T_VCC     |
|       2  | T_SWCLK   |
|       3  | GND       |
|       4  | T_SWDIO   |
|       5  | T_NRST    |
|       6  | T_SWO     |
